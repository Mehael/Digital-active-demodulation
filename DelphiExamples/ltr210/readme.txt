Данный пример демонстрирует работу с модулем LTR210 из среды Delphi.
Пример содержит файл проекта для среды "Delphi 7" (ltr210_delphi.dpr) и для
среды "Embarcadero Delphi XE2" (ltr210_delphi.dproj).

В проекте необходимо указать путь к файлам ltrapi.pas, ltrapidefine.pas,
ltrapitypes.pas, ltr210api.pas новой версии, которые устанавливаются вместе
с библиотеками ltr и необходимы для сборки данного примера
(LTR_INSTALL_DIR/include/pascal2).

Путь можно задать следующим образом:
    Delphi 7    - "Project->Options->Directories/Conditionals->Search path",
    Delphi XE2  - "Project->Options->Delphi Compiler->Search path"

Проект демонстрирует в первую очередь как можно работать с модулем LTR210,
а не как нужно программировать в среде Delphi.

Пример демонстрирует реализацию сбора в покадровом режиме.
Прием кадров выполняется в отдельном потоке, реализованном в классе TLTR210_ProcessThread.
По приему изменяются счетчики принятых пакетов и для примера выводится среднее значение
по каждому каналу.
