Данный пример демонстрирует работу с модулем LTR51 из среды Delphi.
Пример содержит файл проекта для среды "Delphi 7" (ltr51_delphi_interval.dpr) и для
среды "Embarcadero Delphi XE" (ltr51_delphi_interval.dproj).

В проекте необходимо указать путь к файлам ltrapi.pas, ltrapidefine.pas,
ltrapitypes.pas, ltr51api.pas новой версии, которые устанавливаются вместе
с библиотеками ltr и необходимы для сборки данного примера
(LTR_INSTALL_DIR/include/pascal2).

Путь можно задать следующим образом:
    Delphi 7    - "Project->Options->Directories/Conditionals->Search path",
    Delphi XE   - "Project->Options->Delphi Compiler->Search path"

Проект демонстрирует в первую очередь как можно работать с модулем LTR51,
а не как нужно программировать в среде Delphi.

Пример демонстрирует реализацию сбора для заданных настроек.
Файл с прошивкой ПЛИС для LTR51 должен находиться в директории с проектом.
Прием данных выполняется в отдельном потоке, реализованном в классе TLTR51_ProcessThread.

Пример принимает данные от всех разрешенных каналов, настроенных одинаково и на основании принятых
параметров M и N рассчитывает интервалы между заданным количеством фронтов.
При появлении первого фронта для канала просто выводится сообщение, при
появлении заданного кол-ва фронтов рассчитывается время с первого.
При отсутствии заданного кол-ва фронтов за устанровленное макс. время подсчет времени сбрасывается
и снова ожидается первый фронт.
Интервал между двумя фронтами должен быть не менее заданного кол-ва мс.
